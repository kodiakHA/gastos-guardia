# Web Peña Campanar

Aplicacion web para repartir gastos de guardia en Parque Campanar.

## Arrancar en local

```bash
npm start
```

Despues abre:

```text
http://localhost:3000
```

Railway usara el mismo comando de arranque.

## Archivos principales

- `index.html`: aplicacion completa.
- `server.js`: servidor web sin dependencias externas.
- `package.json`: scripts para terminal.
- `railway.json`: configuracion de despliegue en Railway.

## Funciones actuales

- Gastos por almuerzo, pan, comida y cena.
- Control de presentes por guardia.
- Gasto minimo editable.
- Liquidacion directa entre companeros.
- Pantalla limpia para captura de liquidaciones.
- Banner de Bitcoin con precio BTC/EUR desde Binance.
