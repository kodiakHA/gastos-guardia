# Web Peña Campanar

Aplicacion web para repartir gastos de guardia en Parque Campanar.

## Arrancar en local

```bash
npm start
```

Despues abre:

```text
http://localhost:3000
```

Railway usara el mismo comando de arranque.

## Archivos principales

- `index.html`: aplicacion completa.
- `server.js`: servidor web sin dependencias externas.
- `package.json`: scripts para terminal.
- `railway.json`: configuracion de despliegue en Railway.

## Funciones actuales

- Gastos por almuerzo, pan, comida y cena.
- Hasta 3 compradores por almuerzo, comida y cena, con importe separado por comprador.
- Un comprador para pan.
- Control de presentes por guardia.
- Gasto minimo editable.
- Liquidacion directa entre companeros.
- Calendario de guardias con ciclo de 5 dias desde el 29/05/2026.
- Guardado local en el navegador y guardado compartido opcional con Supabase.
- Historial de guardias guardadas.
- Compras previas para guardias futuras: se selecciona la fecha de guardia y queda guardado para liquidar ese dia.
- Pantalla limpia para captura de liquidaciones.
- Banner de Bitcoin con precio BTC/EUR desde Binance.

## Supabase

En Supabase crea una tabla llamada `guardias` con este SQL. La app usa la URL del proyecto y la clave publica `anon`, nunca la clave secreta.

```sql
create table if not exists public.guardias (
  id bigint generated by default as identity primary key,
  guard_date date not null,
  station text not null default 'Parque Campanar',
  payload jsonb not null,
  totals jsonb not null default '{}'::jsonb,
  present_count integer not null default 0,
  updated_at timestamptz not null default now(),
  unique (guard_date, station)
);

alter table public.guardias enable row level security;

create policy "Lectura compartida de guardias"
on public.guardias for select
using (true);

create policy "Crear guardias compartidas"
on public.guardias for insert
with check (true);

create policy "Actualizar guardias compartidas"
on public.guardias for update
using (true)
with check (true);
```

Para que los cambios de otros companeros aparezcan en vivo, activa Realtime para la tabla `guardias` en Supabase. Si no se activa, la app igualmente guarda y carga con los botones.
